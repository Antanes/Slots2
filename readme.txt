
Проект выполнен в програмной среде Visual Studio 2022 Community. Все требования выполнены.


Уровень DAL (Data Access Layer) - логический уровень взаимодействия с базой данных. В asp.net - Class Library. В нем находятся:
- класс ApplicatioDbContext, который описывает взаимодействие с базой данных;
- класс DrinkRepository, в котором содержаться специальные методы для взаимодействия с базой данных;
- Generic интерфейс IBaseRepository, который реализуется в репозитории DrinkRepository.

Уровень Domain. В нем находятся необходимые обЪекты, переменные и общие детали проекта.
- В папке Entity хранится основной объект Drink и класс Global со статическими переменными, которые используются для реализации логики программы;
- Enum - StatusCode здесь находятся коды ошибок и успешного выполнения метода;
- Папка Response, в ней хранится класс BaseResponse, который выступает в качестве возвращаемого объекта для Сервисов (Service->Implementation->DrinkService);
- ViewModels здесь хранится модель представления DrinkViewModel. Класс который служит для получения данных из базы данных и отображения их в представлении.

Уровень Service - это прослойка между базой данных и контроллером. Здесь используются методы репозитория, которые взаимодействуют с базой данных, определенным способом обрабатывают их и возвращают некий результат в контроллер. Здесь содержатся интерфейс IDrinkService и класс DrinkService, который его реализует.

Проект Slots. 
- Папка wwwroot содержит подключаемые библиотеки bootstrap, jquery, изображения и т.д.;
- Папка App_Data содержит в себе файл базы данных Slots.mdf;
- Папка Controllers, в ней находится контроллер DrinkController, который содержит методы действий;
- Папка Views здесь находятся представления и мастер-страница проекта - _Layout. GetDrinks - представление страницы пользователя. GetDrinksAdmin - представление страницы администратора. Save - представление модального окна добавления и изменения напитков;
- Файл параметр appsettings.json, в котором указывается строка подключения к файлу базы данных;
- Класс Initializer инициализирует интерфейсы;
- Класс Program - главный файл приложения, с которого и начинается его выполнение. Код этого файла настраивает и запускает веб-приложение
